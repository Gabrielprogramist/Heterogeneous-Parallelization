## **Практическая работа №6**
**Программирование на OpenCL для CPU и GPU**

---

#### Описание работы

Данная практическая работа посвящена изучению технологии **OpenCL** и её применению для параллельных вычислений на **CPU и GPU**.

В ходе работы были реализованы:

* поэлементное сложение векторов;
* умножение матриц;
* сравнение корректности и времени выполнения.

#### Описание файлов

**`vector_add_opencl.cpp`**
Хост-программа на C++, выполняющая:

* инициализацию OpenCL;
* загрузку kernel-файла;
* запуск вычислений на выбранном устройстве (CPU или GPU);
* замер времени выполнения.

**`kernel_vector_add.cl`**
OpenCL-ядро для поэлементного сложения двух векторов.

**`matrix_mul_opencl.cpp` / `kernel_matrix_mul.cl`**
Реализация умножения матриц с проверкой корректности результатов.
